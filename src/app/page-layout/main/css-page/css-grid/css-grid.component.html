CSS Grid

CSS Grid is a powerful 2-dimensional layout system that allows developers to create complex, responsive designs with clean and maintainable code.

display: grid; behaves like a block-level element by default. That means:

  • It takes up the full width of its parent container.
  • It starts on a new line, pushing other content below.
  • You can apply block-level styles like width, margin, padding, etc.

grid-auto-flow:

When you set display: grid; on a container, the grid is created with one column and as many rows as needed to fit the child elements.That's because we have grid-auto-flow: row; by defolt.

If you change the defolt value
grid-auto-flow: row; to
grid-auto-flow: column;
as a result you will see the grid filling by columns instead of rows.

You can also use display: inline-grid; which behaves like an inline-block element. In this case, the grid cells will adjust to the size of their content.
