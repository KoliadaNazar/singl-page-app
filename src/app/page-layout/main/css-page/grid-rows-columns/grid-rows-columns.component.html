Grid Lines

  • The lines between columns are called column lines.
  • The lines between rows are called row lines.

Defining Rows and Columns.

  • grid-template-columns: defines the number and size of columns.
  • grid-template-rows: defines the number and size of rows.
  • The spaces between each column/row are called gaps.

.grid {
display: grid;
column-gap: 18px;
row-gap: 18px;
grid-template-columns: repeat(3, minmax(60px, 100px));
grid-template-rows: repeat(3, 100px);
}

Grid Aligning

1. Aligning the Entire Grid
Works only when the grid is smaller than the container’s width.

justify-content - Horizontal Alignment
align-content - Vertical Alignment
place-content - is a shorthand property for both

Values:

  • stretch: stretches the grid to fill the container (default if possible)
  • start: aligns the grid to the left
  • end: aligns the grid to the right
  • center: centers the grid horizontally
  • space-between: equal space between grid tracks
  • space-around: equal space around each track
  • space-evenly: equal space between and around all tracks

.grid {
height: 440px;
max-width: 440px;
display: grid;
grid-template-columns: repeat(3, minmax(4px, 80px));
grid-template-rows: repeat(3, 80px);
justify-content: center;
align-content: center;
}

2. Aligning Items in the Grid
Controls how items are aligned within their individual grid areas.

justify-items - Horizontal alignment of items inside their cells
align-items - Vertical alignment of items inside their cells
place-items - is a shorthand property for both

Values:

  • stretch: stretches the item to fill the grid area (default)
  • start: aligns the item to the start of the area
  • end: aligns the item to the end of the area
  • center: centers the item inside the area

.grid {
height: 440px;
max-width: 440px;
display: grid;
grid-template-columns: repeat(2, minmax(40px, 120px));
grid-template-rows: repeat(2, 120px);
place-content: center;
align-items: center;
justify-items: end;
}

3. Aligning Individual Items Inside Their Cells
Used to override alignment for a single grid item.

justify-self - Horizontal alignment of an individual item
align-self - Vertical alignment of an individual item
place-self - is a shorthand property for both

Values:

  • stretch: stretches the item to fill the grid area (default)
  • start: aligns the item to the start of the area
  • end: aligns the item to the end of the area
  • center: centers the item inside the area

.grid {
display: grid;
grid-template-columns: repeat(2, minmax(40px, 120px));
grid-template-rows: repeat(2, 120px);
place-content: center;
}

.grid__item:nth-child(1){
align-self: start;
justify-self: start;
}

.grid__item:nth-child(2){
align-self: center;
justify-self: center;
}

.grid__item:nth-child(3){
align-self: start;
justify-self: center;
}

.grid__item:nth-child(4){
align-self: end;
justify-self: end;
}

Placing Grid Items
Manually place items within the grid using line numbers or named areas.

  • grid-column-start / grid-column-end – place item horizontally
  • grid-column - shorthand for grid-column-start and grid-column-end
  • grid-row-start / grid-row-end – place item vertically
  • grid-row:shorthand for grid-row-start and grid-row-end
  • grid-area: The grid-area property is a shorthand for: 
          grid-row-start 
    
      grid-column-start 
    
      grid-row-end 
    
      grid-column-end

.grid {
max-width: 420px;
display: grid;
grid-template-columns: repeat(3, minmax(40px, 120px));
grid-template-rows: repeat(2, 120px);

Named Areas. Advanced

grid-template-areas – defines named layout areas in the grid
grid-area – assigns a grid item to a named area

Syntax:

  • grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer"
  • Each string represents a row; names define the area cells.
  • Use "." for empty cells with no name.

.grida-area {
max-width: 600px;
display: grid;
gap: 18px;
grid-template-rows: 50px 250px 50px;
grid-template-columns: 10px minmax(20px, 110px) 2fr 10px;
grid-template-areas:
'. header header .'
'. sidebar main .'
'. footer footer .';

}
.grid__area-item:nth-child(1){
grid-area: header;
}

.grid__area-item:nth-child(2){
grid-area: sidebar;
(grid-area: row-start / column-start / row-end / column-end)
}

.grid__area-item:nth-child(3){
grid-area: main;
(grid-area: row-start / column-start / row-end / column-end)
}

.grid__area-item:nth-child(4){
grid-area: footer;
(grid-area: row-start / column-start / row-end / column-end)
