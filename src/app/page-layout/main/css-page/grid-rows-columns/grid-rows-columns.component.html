Grid Lines

  • The lines between columns are called column lines.
  • The lines between rows are called row lines.

Defining Rows and Columns.

  • grid-template-columns: defines the number and size of columns.
  • grid-template-rows: defines the number and size of rows.
  • The spaces between each column/row are called gaps.

.grid {
max-width: 380px;
display: grid;
column-gap: 18px;
row-gap: 18px;
grid-template-columns: repeat(3, minmax(60px, 100px));
grid-template-rows: repeat(3, 100px);
}

Grid Aligning

1. Aligning the Entire Grid
Works only when the grid is smaller than the containerâ€™s width.

justify-content - Horizontal Alignment
align-content - Vertical Alignment
place-content - is a shorthand property for both

Values:

  • stretch: stretches the grid to fill the container (default if possible)
  • start: aligns the grid to the left
  • end: aligns the grid to the right
  • center: centers the grid horizontally
  • space-between: equal space between grid tracks
  • space-around: equal space around each track
  • space-evenly: equal space between and around all tracks

.grid {
height: 440px;
max-width: 440px;
display: grid;
grid-template-columns: repeat(3, minmax(4px, 80px));
grid-template-rows: repeat(3, 80px);
justify-content: center;
align-content: center;
}

2. Aligning Items in the Grid
Controls how items are aligned within their individual grid areas.

justify-items - Horizontal alignment of items inside their cells
align-items - Vertical alignment of items inside their cells
place-items - is a shorthand property for both

Values:

  • stretch: stretches the item to fill the grid area (default)
  • start: aligns the item to the start of the area
  • end: aligns the item to the end of the area
  • center: centers the item inside the area

.grid {
height: 440px;
max-width: 440px;
display: grid;
grid-template-columns: repeat(2, minmax(40px, 120px));
grid-template-rows: repeat(2, 120px);
place-content: center;
align-items: center;
justify-items: end;
}

3. Aligning Individual Items Inside Their Cells
Used to override alignment for a single grid item.

justify-self - Horizontal alignment of an individual item
align-self - Vertical alignment of an individual item
place-self - is a shorthand property for both

Values:

  • stretch: stretches the item to fill the grid area (default)
  • start: aligns the item to the start of the area
  • end: aligns the item to the end of the area
  • center: centers the item inside the area

.grid {
display: grid;
grid-template-columns: repeat(2, minmax(40px, 120px));
grid-template-rows: repeat(2, 120px);
place-content: center;
}

.grid__item:nth-child(1){
align-self: start;
justify-self: start;
}

.grid__:nth-child(2){
align-self: center;
justify-self: center;
}

.grid__:nth-child(3){
align-self: start;
justify-self: center;
}

.grid__:nth-child(3){
align-self: end;
justify-self: end;
}
