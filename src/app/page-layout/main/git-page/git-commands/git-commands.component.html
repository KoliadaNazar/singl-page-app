Git Commands

Basic Commands

# Check installed Git version
git --version

# Set user name and email globally
git config --global "user.name"
git config --global "user.email"

#Initializes a new Git repository
git init

#Clones a remote repository
git clone URL

#Shows the current state of the
working directory and staging area
git status

#Adds all changes for the next commit
git add .

# Track a specific file
git add "filename.txt"

#Commits staged changes with message
git commit -m "message"

#Push local commits to remote branch
git push origin "branch"

#Fetches and merges changes from
a remote branch (fetch + merge)
git pull origin "branch"

#Downloads changes from remote,
but doesnâ€™t apply them
git fetch origin

Branching & Merging

# Lists local branches
git branch

# Creates a new branch
git branch "name"

#Switches to the specified branch
git checkout "branch"

#Creates and switches to a new branch
git checkout -b "branch"

#Merges the specified branch
into the current one
git merge "branch"

#Reapplies commits on top of
another base branch (clean history)
git rebase "branch"

# Deletes a local branch
git branch -d "name"

#Deletes a remote branch
git push origin --delete "name"

Inspection & History

# Shows commit history
git log

#Compact version of commit history
git log --oneline

#Shows differences between working
directory and last commit
git diff

# Shows difference between two commits
git diff "commit1" "commit2"

#Shows who modified each line
in a file and when
git blame "file"

Fixing Mistakes & Stashing

# Temporarily stores uncommitted changes
git stash

#Restores stashed changes
git stash apply

#Uncommits the last commit, keeps changes
in filedirectory and last commit
git reset HEAD~1git reset HEAD~1

#Discards changes in a specific file
git checkout -- "file"

#Removes untracked files
git clean -f

#Shows a log of all Git actions
git reflog
