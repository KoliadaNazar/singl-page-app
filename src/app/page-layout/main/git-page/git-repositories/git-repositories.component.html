Local Repository

  • The copy of the project on your own computer, including your working directory and .git folder.

Remote Repository

  • А version of the repository hosted on a server (e.g., GitHub, GitLab, Bitbucket) that others can access and collaborate on.

Bare Repositor

  • А repository that contains only the Git version history (no working directory); used mainly as a central repo for collaboration (typically on servers).
