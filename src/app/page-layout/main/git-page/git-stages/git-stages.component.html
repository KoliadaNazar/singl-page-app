Git workflow stages

There are several stages your code goes through during development:

  • Working Directory — the actual files in your project folder that you're editing.
  • Staging Area (Index) — the place where changes are added using #git add before committing.
  • Local Repository – the database on your computer are stored with #git commit.
  • Remote Repository – hosted on platforms like GitHub,GitLab,Bitbucket updated via #git push.
