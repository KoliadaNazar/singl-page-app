display: grid; behaves like a block-level element by default. That means:

  • It takes up the full width of its parent container.
  • It starts on a new line, pushing other content below.
  • You can apply block-level styles like width, margin, padding, etc.

grid-auto-flow:

When you set display: grid; on a container, the grid is created with one column and as many rows as needed to fit the child elements.That's because we have grid-auto-flow: row; by defolt.
