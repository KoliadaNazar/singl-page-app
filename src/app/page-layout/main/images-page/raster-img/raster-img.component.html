Raster Graphics (Bitmap Images)

Definition: Composed of a grid of pixels, each with a specific color value.

Key Features:

  • Fixed resolution (quality loss when scaled up).
  • Ideal for complex images like photos
  • Larger file sizes compared to vectors.

Common Formats:

  • JPEG – Best for photographs (lossy compression).
  • GIF – Supports animation (limited colors).
  • WebP – Modern alternative (better compression than JPEG/PNG).
  • AVIF – Next-gen format (superior compression, supports HDR & transparency).
  • HEIF/HEIC – Apple’s preferred format (stores images & sequences efficiently).

Use Cases:

  • Digital photography
  • Web images (banners, backgrounds)
  • Textures in games

Raster Graphics Summary

  • Pixel-based: Made of a grid of tiny squares (pixels), each with its own color.
  • Resolution-dependent: Scaling up causes blurriness or pixelation
  • Best for: Photographs, realistic textures, complex visuals.
  • Common formats: JPEG, PNG, GIF, WebP,AVIF,HEIF/HEIC .
  • Pros: Rich detail, widely supported, easy to edit.
  • Cons: Not scalable, larger file sizes for high-res images.
  • HEIF/HEIC – Apple’s preferred format (stores images & sequences efficiently).

In frontend development, raster graphics are great for detailed visuals like photos. Since they’re resolution-dependent and can slow down pages, use theright format, compress images, and enable lazy loading and responsive sizes to keep performance high.
